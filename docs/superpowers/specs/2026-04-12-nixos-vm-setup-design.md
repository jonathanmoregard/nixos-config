# NixOS VM Setup — Design Spec

**Date:** 2026-04-12
**Author:** jonathanmoregard

## Goal

Set up NixOS in a QEMU/KVM VM on the current Linux laptop, with Claude Code installed, all config in a private GitHub repo. The repo is structured to also accommodate a Mac Mini (M-series, 64GB) running nix-darwin for AI inference when it arrives.

## Constraints

- Host machine: ~3.1 GB RAM available, Intel i5-1240P
- VM allocation: 2 GB RAM (leaves ~1 GB for host)
- VM tool: QEMU/KVM (libvirt/virt-manager)
- GitHub: private repo `jonathanmoregard/nixos-config`

## Repo Structure

```
nixos-config/
├── flake.nix                          # outputs: nixosConfigurations.vm + darwinConfigurations.mac-mini
├── flake.lock
├── hosts/
│   ├── vm/
│   │   ├── default.nix                # VM hardware, boot, networking
│   │   └── hardware-configuration.nix # auto-generated by nixos-generate-config
│   └── mac-mini/
│       └── default.nix                # placeholder — nix-darwin, Ollama, launchd
├── modules/
│   ├── common.nix                     # claude-code, git, shared CLI tools
│   ├── nixos/
│   │   └── vm-tweaks.nix              # low-RAM tuning: max-jobs=1, zram/swap
│   └── darwin/
│       └── inference.nix              # Ollama + MLX, Metal, launchd service
└── home/
    └── jonathan.nix                   # Home Manager: zsh, git identity, dotfiles
docs/
└── superpowers/specs/
    └── 2026-04-12-nixos-vm-setup-design.md
```

## flake.nix Inputs

```nix
inputs = {
  nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
  home-manager.url = "github:nix-community/home-manager";
  home-manager.inputs.nixpkgs.follows = "nixpkgs";
  nix-darwin.url = "github:LnL7/nix-darwin";
  nix-darwin.inputs.nixpkgs.follows = "nixpkgs";
};
```

## NixOS VM (`hosts/vm/`)

- **NixOS channel:** nixos-unstable
- **Boot:** GRUB, single partition (or simple GPT for virt)
- **Networking:** NetworkManager, NAT via libvirt default bridge
- **Users:** `jonathan` with sudo, SSH key auth
- **Display:** headless or minimal — VM used via SSH from host

### Low-RAM Tuning (`modules/nixos/vm-tweaks.nix`)

```nix
nix.settings = {
  max-jobs = 1;
  cores = 1;
};
# Enable zram swap to handle nix builds without OOM
zramSwap.enable = true;
```

## Shared Modules (`modules/common.nix`)

Packages applied to both VM and Mac Mini:
- `pkgs.claude-code` (available in nixpkgs unstable)
- `pkgs.git`
- `pkgs.gh` (GitHub CLI)
- `pkgs.ripgrep`, `pkgs.fd`, `pkgs.jq`

## Home Manager (`home/jonathan.nix`)

- `programs.zsh.enable = true` with autosuggestions + syntax highlighting
- `programs.git` with name/email
- `home.stateVersion = "25.11"`
- Applied via `home-manager.nixosModules.home-manager` in VM flake output

## Mac Mini Placeholder (`hosts/mac-mini/`)

Committed as a skeleton now, fleshed out on arrival:
- nix-darwin system config
- `modules/darwin/inference.nix`: Ollama (Metal-accelerated), MLX Python env, launchd service
- 64 GB unified memory → can run 70B models at Q4 comfortably
- Metal GPU used automatically by Ollama; ANE accessible via MLX/Core ML

## GitHub Setup

1. `git init` in repo root
2. Create private repo via `gh repo create jonathanmoregard/nixos-config --private`
3. Push initial commit with design doc + skeleton config

## Implementation Order

1. Git init + GitHub remote
2. Write `flake.nix` skeleton (both outputs declared)
3. Write `modules/common.nix` + `home/jonathan.nix`
4. Write `hosts/mac-mini/default.nix` placeholder
5. Create NixOS VM in virt-manager (2 GB RAM, 20 GB disk, NAT)
6. Boot NixOS installer ISO, run `nixos-generate-config`
7. Copy `hardware-configuration.nix` into repo
8. Write `hosts/vm/default.nix` + `modules/nixos/vm-tweaks.nix`
9. `nixos-install --flake .#vm`
10. Verify Claude Code works in VM
11. Push final config to GitHub
